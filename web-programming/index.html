핵심 암기 모음
CSR vs SSR
CSR = 빈 HTML + JS 렌더링 / SSR = 완성 HTML 전달
TCP vs UDP
TCP = 안전·순서 / UDP = 빠름·비보장
소켓 1 vs 3
소켓 1 = 대기 전용 / 소켓 3 = 클라이언트 전용 통신
큐 우선순위
마이크로태스크(Promise) > 태스크(setTimeout)
const vs let
const = 재할당 불가 (내부 수정 가능) / let = 재할당 가능
typeof null
"object" — JS 설계 버그, null은 원시 타입
