싱글스레드 vs 웹 워커
싱글스레드

• 이벤트 루프로 처리
웹 워커
• 별도 스레드 생성
• DOM 접근 불가
• 메인 블로킹 없음
⚠️ 콜스택 블로킹: 초반 무거운 작업 → 이후 콜백 모두 밀림
