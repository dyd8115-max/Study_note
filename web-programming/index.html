Web 07 Asynchronous Programming
동기 vs 비동기
동기
• 순서대로
• 대기
• ❌ 블로킹
비동기
• 백그라운드
• 기다리지 않음
• ✅ 논블로킹
Promise 3가지 상태
상태메서드설명
Fulfilled.then()resolve()
Rejected.catch()reject()
항상.finally()성공/실패 무관
⚠️ Promise는 한 번 상태가 변경되면 다시 변할 수 없음!
Fetch & Promise
fetch(url) .then(res => res.json()) .then(data => console.log(data))
response.json()도 Promise 반환! 바디 크기가 크면 파싱에 시간 소요
async/await
async function getData() { const data = await fetch(url); }
async: 함수 비동기화 · Promise 반환
await: Promise settle 대기 · async 내에서만
