CH01 · 벡터 기초
크기 (Norm) ‖v‖ = √(x₁²+x₂²+…+xₙ²)
단위벡터 u = v / ‖v‖
내적 A·B = ΣAᵢBᵢ = ‖A‖‖B‖cosθ
외적 크기 ‖A×B‖ = ‖A‖‖B‖sinθ
내적값의미
양수같은 방향
0직교 (90°)
음수반대 방향
CH01 · 외적 계산
A×B = | i j k | | A₁ A₂ A₃ | | B₁ B₂ B₃ |
i 성분A₂B₃ − A₃B₂
j 성분−(A₁B₃ − A₃B₁)
k 성분A₁B₂ − A₂B₁

예) [2,3,4]×[5,6,7] = [-3, 6, -3]

CH01 · 행렬 연산
행렬곱 Cᵢⱼ = Σₗ AᵢₗBₗⱼ
역행렬 A⁻¹ = 1/det(A) · [d -b; -c a]
det(A) = ad − bc
rank = n역행렬 존재
rank < n역행렬 없음

rank = 행 축소 후 0이 아닌 행의 수

CH01 · 투영 & 선형변환
투영 proj(x) = (x·u)·u
잔차 r = x − proj(x)
선형변환 y = Wx
M < N차원 축소
M > N차원 확대

기저 조건: 선형 독립 + 벡터 수 = 공간의 차원

CH02 · 확률 공간
기호이름설명
ω근원 사건실험 결과 하나
Ω표본 공간모든 ω의 집합
A사건Ω의 부분집합
Fσ-대수확률 계산 가능한 사건 모음
Kolmogorov: P(A)≥0, P(Ω)=1, P(A∪B)=P(A)+P(B) if 배반
CH02 · CDF / PMF / PDF
CDF F(x) = P(X ≤ x)
PMF p(x) = P(X=x), Σp(x) = 1 [이산]
PDF P(a≤X≤b) = ∫f(x)dx, ∫f(x)dx = 1 [연속]
구간 확률P(a<X≤b) = F(b)−F(a)

연속: P(X=x) = 0 → 반드시 구간으로 계산

CH02 · 조건부 / 독립 / 베이즈
조건부 P(A|B) = P(A∩B) / P(B)
독립 P(A∩B) = P(A)·P(B)
베이즈 P(θ|X) = P(X|θ)·P(θ) / P(X)
연쇄 법칙P(A∩B) = P(A|B)·P(B)

Prior + 데이터 → 사후 확률 업데이트

CH02 · 기댓값 / 분산
E[X] = Σ x·p(x) [이산]
E[X] = ∫ x·f(x)dx [연속]
Var[X] = E[X²] − (E[X])²
SD[X] = √Var[X]
선형성E[aX+b] = aE[X]+b
스케일Var[aX+b] = a²Var[X]
CH03 · 통계 파라미터
평균 x̄ = (1/n) Σxᵢ
분산 s² = 1/(n-1) Σ(xᵢ−x̄)²
표준편차 s = √s²
공분산 C(x,y) = 1/(n-1) Σ(xᵢ−x̄)(yᵢ−ȳ)
상관계수 ρ = C(x,y) / (σₓσᵧ), |ρ| ≤ 1
CH03 · 왜도 / 첨도 / Bias-Variance
왜도 = Σ(xᵢ−x̄)³ / [Σ(xᵢ−x̄)²]^(3/2)
첨도 = Σ(xᵢ−x̄)⁴ / [Σ(xᵢ−x̄)²]²
왜도 0좌우 대칭
첨도 3정규분포
E[(y−ŷ)²] = Bias² + Variance + Noise

모델 복잡도 ↑ → Bias ↓, Variance ↑ (Trade-off)

CH03 · 정규분포 / CLT / 회귀
정규분포 N(x;μ,σ²) = 1/√(2πσ²) · e^(-(x-μ)²/2σ²)
CLT X̄ₙ → N(μ, σ²/n) as n→∞
회귀 ŷ = θ₀ + θ₁x
MSE J(θ) = (1/N) Σ(ŷᵢ−yᵢ)²
정규방정식 θ = (XᵀX)⁻¹Xᵀy

68-95-99.7 규칙: P(μ±σ)≈68%, P(μ±2σ)≈95%

CH03 · MLE / MAP
우도 L(θ;X) = Πᵢ p(xᵢ|θ) [i.i.d.]
로그우도 ℓ(θ) = Σᵢ log p(xᵢ|θ)
MLE θ̂ = argmax ℓ(θ)
MAP θ̂ = argmax [ℓ(θ) + log P(θ)]
Gaussian PriorL2 정규화 (Ridge)
Laplace PriorL1 정규화 (Lasso)

Gaussian MLE: μ̄ = (1/n)Σxᵢ, σ̄² = (1/n)Σ(xᵢ−μ̄)²

주요 확률 분포 요약
