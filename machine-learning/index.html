CH01 · 벡터 기초
크기 (Norm) ‖v‖ = √(x₁²+x₂²+…+xₙ²)
단위벡터 u = v / ‖v‖
내적 A·B = ΣAᵢBᵢ = ‖A‖‖B‖cosθ
외적 크기 ‖A×B‖ = ‖A‖‖B‖sinθ
내적값의미
양수같은 방향
0직교 (90°) — a⊥b
음수반대 방향

직교 조건: a·b = 0 ↔ cosθ = 0

CH01 · 평면 & 법선 벡터
평면 P = {αu₁ + βu₂ | α,β ∈ ℝ}
조건의미
a·u₁ = 0 이고 a·u₂ = 0법선 벡터 — P에 수직
b = αu₁ + βu₂평면 위 — 선형결합
법선 ⊥ 평면 위 벡터a·b = 0 반드시 성립

법선 벡터는 평면 위의 모든 벡터와 수직이므로 내적=0

CH01 · 외적 계산
A×B = | i j k | | A₁ A₂ A₃ | | B₁ B₂ B₃ |
i 성분A₂B₃ − A₃B₂
j 성분−(A₁B₃ − A₃B₁)
k 성분A₁B₂ − A₂B₁

예) [2,3,4]×[5,6,7] = [-3, 6, -3]

CH01 · 행렬 연산
행렬곱 Cᵢⱼ = Σₗ AᵢₗBₗⱼ
역행렬 A⁻¹ = 1/det(A) · [d -b; -c a]
det(A) = ad − bc
rank = n역행렬 존재
rank < n역행렬 없음

rank = 행 축소 후 0이 아닌 행의 수

CH01 · 투영 & 선형변환
투영 proj(x) = (x·u)·u
잔차 r = x − proj(x)
선형변환 y = Wx
M < N차원 축소
M > N차원 확대

기저 조건: 선형 독립 + 벡터 수 = 공간의 차원

CH02 · 확률 공간
기호이름설명
ω근원 사건실험 결과 하나
Ω표본 공간모든 ω의 집합
A사건Ω의 부분집합
Fσ-대수확률 계산 가능한 사건 모음
Kolmogorov: P(A)≥0, P(Ω)=1, P(A∪B)=P(A)+P(B) if 배반
CH02 · CDF / PMF / PDF
CDF F(x) = P(X ≤ x)
PMF p(x) = P(X=x), Σp(x) = 1 [이산]
PDF P(a≤X≤b) = ∫f(x)dx, ∫f(x)dx = 1 [연속]
구간 확률P(a<X≤b) = F(b)−F(a)

연속: P(X=x) = 0 → 반드시 구간으로 계산

CH02 · 조건부 / 독립 / 베이즈
조건부 P(A|B) = P(A∩B) / P(B)
독립 P(A∩B) = P(A)·P(B)
베이즈 P(θ|X) = P(X|θ)·P(θ) / P(X)
연쇄 법칙P(A∩B) = P(A|B)·P(B)

Prior + 데이터 → 사후 확률 업데이트

CH02 · 기댓값 / 분산
E[X] = Σ x·p(x) [이산]
E[X] = ∫ x·f(x)dx [연속]
Var[X] = E[X²] − (E[X])²
SD[X] = √Var[X]
선형성E[aX+b] = aE[X]+b
스케일Var[aX+b] = a²Var[X]
CH03 · 통계 파라미터
평균 x̄ = (1/N) Σxᵢ
분산 σ² = (1/N) Σ(xᵢ−x̄)² [MLE, 편향]
불편 s² = 1/(N-1) Σ(xᵢ−x̄)²
공분산 Cov = (1/N) Σ(xᵢ−x̄)(yᵢ−ȳ)
상관계수 r = Cov(X,Y) / (σₓσᵧ), |r|≤1

MLE σ²은 편향 → μ̂ 추정으로 자유도 1 감소 → N-1 사용해야 불편

CH03 · 이항 분포 & 정규 근사
X ~ B(n,p) → E[X]=np, Var(X)=np(1-p)
CLT 근사: Z = (X−np) / √(np(1−p)) ≈ N(0,1)
이항 MLEp̂ = k/n (관측 빈도)
CLT 조건n ≥ 30 (충분히 클 때)

68-95-99.7 규칙: P(μ±σ)≈68%, P(μ±2σ)≈95%

CH03 · 왜도 / 첨도 / Bias-Variance
왜도 = Σ(xᵢ−x̄)³ / [Σ(xᵢ−x̄)²]^(3/2)
첨도 = Σ(xᵢ−x̄)⁴ / [Σ(xᵢ−x̄)²]²
왜도 0좌우 대칭
첨도 3정규분포
E[(y−ŷ)²] = Bias² + Variance + Noise

복잡도↑ → Bias↓, Variance↑ (Trade-off)

CH03 · MLE / MAP
우도 L(θ;X) = Πᵢ p(xᵢ|θ) [i.i.d.]
로그우도 ℓ(θ) = Σᵢ log p(xᵢ|θ)
MLE θ̂ = argmax ℓ(θ)
MAP θ̂ = argmax [ℓ(θ) + log P(θ)]
Gaussian PriorL2 정규화 (Ridge)
Laplace PriorL1 정규화 (Lasso)

가우시안 MLE: μ̂ = x̄, σ̂² = (1/N)Σ(xᵢ−μ̂)² [편향됨]

CH03 · 가우시안 분류 / LDA
P(x|Cₖ) = 1/√(2πσ²) · exp(−(x−μₖ)²/2σ²)
로그우도비 = 1/(2σ²) · [−(x−μ₁)² + (x−μ₂)²]
결정 경계x* = (μ₁+μ₂)/2
x² 소거→ 선형 결정 경계
공통 분산σ̂² = (Σ편차²₁ + Σ편차²₂) / (n₁+n₂)

등 사전 확률 + 공통 분산 조건에서만 중점이 결정 경계

CH03 · 라그랑주 승수법 (Categorical MLE)
ℓ = Σ nₖ log πₖ, 제약: Σπₖ = 1
L(π,λ) = ℓ − λ(Σπₖ − 1)
  • ∂L/∂πₖ = 0 → πₖ = nₖ/λ
  • 제약 대입 → λ = n (전체 샘플 수)
  • π̂ₖ = nₖ/n (경험적 확률)

결과 = Softmax + Cross-Entropy로 학습한 딥러닝의 수렴값과 동일

CH04 · 선형 회귀 (MSE 최소화)
ŷ = θ₁x + θ₀
J(θ) = (1/N) Σ(θ₁xᵢ+θ₀−yᵢ)²
∂J/∂θ₀=0θ₁Σxᵢ + Nθ₀ = Σyᵢ
∂J/∂θ₁=0θ₁Σxᵢ² + θ₀Σxᵢ = Σxᵢyᵢ
정규방정식 θ = (XᵀX)⁻¹Xᵀy

볼록 함수 → 편미분=0이 전역 최솟값 / 직선은 항상 (x̄, ȳ) 통과

CH05 · 로지스틱 회귀
σ(z) = 1/(1+e⁻ᶻ), σ'(z) = σ(z)(1−σ(z))
BCE L = −[y log ŷ + (1−y) log(1−ŷ)]
∂L/∂z = ŷ − y (시그모이드+로그 상쇄)
∂J/∂θ₀(1/N) Σ(ŷᵢ−yᵢ)
∂J/∂θ₁(1/N) Σ xᵢ(ŷᵢ−yᵢ)
경사하강법θ ← θ − α·∂J/∂θ

초기 θ=0 → 모든 ŷ=0.5 → J = log2 ≈ 0.693 (최대 불확실성)

CH07 · KNN (K-Nearest Neighbors)
유클리드 d = √(Σ(xᵢ−zᵢ)²)
맨해튼 d = Σ|xᵢ−zᵢ|
  • 훈련 데이터 메모리 저장 (Lazy Learning)
  • 쿼리와 모든 점 거리 계산 — O(n·d)
  • 가장 가까운 K개 선택
  • 다수결(분류) / 평균(회귀)
K=1과적합 (낮은 편향, 높은 분산)
K=n과소적합 (높은 편향, 낮은 분산)

차원의 저주: 고차원 → 거리 집중 → PCA로 선처리 필요 / 스케일 정규화 필수

CH07 · K-means 클러스터링
J = Σₖ Σ_{xᵢ∈Cₖ} ‖xᵢ−μₖ‖² → min
  • K개 초기 중심 선택 (K-means++권장)
  • 할당: cᵢ = argminₖ ‖xᵢ−μₖ‖²
  • 갱신: μₖ = 평균(Cₖ)
  • 할당 변화 없을 때까지 2~3 반복
K 선택Elbow Method / Silhouette Score
수렴 보장J 단조 감소 + 유한 경우의 수

전역 최솟값 비보장 → n_init으로 여러 번 실행 후 최소 J 선택
