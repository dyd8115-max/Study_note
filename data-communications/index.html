CH03 · 전송률 한계 공식
나이퀴스트 (잡음 없음)
Bit Rate = 2 × BW × log₂L
섀논 (잡음 있음)
C = BW × log₂(1 + SNR)
SNR_dB = 10 × log₁₀(SNR)
감쇠 dB = 10 × log₁₀(P₂/P₁)

1+SNR 이유: SNR=0일 때 log₂(0)=∞ 방지 → C=0으로 자연스럽게 처리

실제 설계: 섀논으로 상한 → 나이퀴스트로 신호 준위 수 계산

CH04 · PCM (아날로그→디지털) — 표양부
  • 표본화 (Sampling) — 일정 간격으로 진폭 측정
    나이퀴스트: 표본화율 ≥ 최대주파수 × 2
    예) 음성 4,000Hz → 8,000 samples/s
  • 양자화 (Quantizing) — 실수값을 정수로 근사
    양자화 잡음 발생 (근사 오차)
  • 부호화 (Encoding) — n비트 코드로 변환
Δ = (V_max − V_min) / L
Δ(delta): 구간 크기 · V_max/V_min: 최대/최소 전압 · L(levels): 준위 수
L↑ → Δ↓ → 더 세밀한 근사 → 양자화 잡음 감소
Bit rate = f_s × n_b
f_s(sampling frequency): 1초당 표본화 횟수 · n_b(number of bits): 표본당 비트 수

예) 음성: 8,000 × 8bit = 64,000 bps

PCM → 광대역: 아날로그→디지털(PCM) → 디지털→아날로그 변조(광대역) → 전송 (VoIP)
