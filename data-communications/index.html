CH 01 선형대수 — 벡터 · 행렬 · 변환
벡터 기초 공식
크기 (Norm) ‖v‖ = √(v₁²+v₂²+…+vₙ²)
단위벡터 û = v / ‖v‖
내적 A·B = ΣAᵢBᵢ = ‖A‖‖B‖cosθ
외적 크기 ‖A×B‖ = ‖A‖‖B‖sinθ
내적 부호
양수 같은 방향
0 직교 (90°)
음수 반대 방향
직교 조건
a·b = 0
↔ cosθ = 0
↔ θ = 90°
외적 (Cross Product) 계산
A×B = | i j k | | A₁ A₂ A₃ | | B₁ B₂ B₃ |
i 성분A₂B₃ − A₃B₂
j 성분−(A₁B₃ − A₃B₁)
k 성분A₁B₂ − A₂B₁

예) [2,3,4]×[5,6,7] = [-3, 6, -3]

행렬 연산 & 역행렬
행렬곱 Cᵢⱼ = Σₗ AᵢₗBₗⱼ
det(A) = ad − bc (2×2)
A⁻¹ = 1/det(A) · [ d -b ] [-c a ]
rank = n역행렬 존재
rank < n역행렬 없음
rank 계산행 축소 후 0이 아닌 행 수
투영 & 선형변환
proj(x) = (x·û)·û [단위벡터 û에 투영]
잔차 r = x − proj(x) [투영 후 남은 성분]
선형변환 y = Wx [W: M×N 행렬]
차원 변환
M < N → 축소
M > N → 확대
기저 조건
선형 독립
+ 벡터 수 = 차원
✦ 연습문제
Q1
A=[1,2,3], B=[4,0,-2]일 때 내적 A·B와 두 벡터의 관계는?
A·B = 4+0−6 = −2 → 음수 → 둔각(반대 방향 성분)
Q2
2×2 행렬 A=[[3,1],[5,2]]의 역행렬을 구하라.
det=3·2−1·5=1 → A⁻¹=[[2,-1],[-5,3]]
Q3
x=[3,4], û=[1,0]일 때 투영과 잔차는?
proj=(3·1)·[1,0]=[3,0], 잔차=[0,4]
CH 02 확률 이론 — 공간 · 분포 · 베이즈
확률 공간 (Ω, F, P)
기호이름설명
Ω표본 공간모든 결과의 집합
ω근원 사건실험 결과 하나
A사건Ω의 부분집합
Fσ-대수확률 계산 가능한 사건 모음
Kolmogorov: P(A)≥0, P(Ω)=1 P(A∪B)=P(A)+P(B) [A,B 배반]
CDF / PMF / PDF
CDF F(x) = P(X ≤ x) [이산·연속 공통]
PMF p(x) = P(X=x), Σp(x)=1 [이산]
PDF P(a≤X≤b) = ∫ₐᵇ f(x)dx [연속]
구간 P(a<X≤b) = F(b) − F(a)

연속: P(X=x)=0 → 반드시 구간으로 계산

조건부 · 독립 · 베이즈
조건부 P(A|B) = P(A∩B) / P(B)
독립 P(A∩B) = P(A)·P(B)
연쇄 P(A∩B) = P(A|B)·P(B)
베이즈 P(θ|X) = P(X|θ)·P(θ) / P(X)
Prior
데이터 관측 전
θ에 대한 믿음
Posterior
데이터 후
갱신된 믿음
기댓값 · 분산
E[X] = Σ x·p(x) [이산]
E[X] = ∫ x·f(x)dx [연속]
Var[X] = E[X²] − (E[X])²
선형성E[aX+b] = aE[X]+b
스케일Var[aX+b] = a²Var[X]
독립 합Var[X+Y] = Var[X]+Var[Y]
✦ 연습문제
Q1
X의 PMF: P(0)=0.5, P(1)=0.3, P(2)=0.15, P(3)=0.05일 때 E[X]와 Var[X]를 구하라.
E[X]=0.75, E[X²]=1.35, Var[X]=1.35−0.5625=0.7875
Q2
P(비)=0.3, P(흐림|비)=0.9, P(흐림|맑음)=0.4일 때 P(비|흐림)은?
P(흐림)=0.9×0.3+0.4×0.7=0.55, P(비|흐림)=0.27/0.55≈0.49
CH 03 통계 & MLE / MAP — 추정 이론
통계 파라미터
평균 x̄ = (1/N) Σxᵢ
분산(편향) σ² = (1/N) Σ(xᵢ−x̄)² ← MLE
분산(불편) s² = 1/(N-1) Σ(xᵢ−x̄)²
공분산 Cov = (1/N) Σ(xᵢ−x̄)(yᵢ−ȳ)
상관계수 ρ = Cov(X,Y)/(σₓσᵧ), |ρ|≤1

MLE σ² 편향 이유: μ̂ 추정으로 자유도 1 감소 → N-1 사용해야 불편 추정량

왜도 · 첨도 · Bias-Variance
왜도 = Σ(xᵢ−x̄)³ / [Σ(xᵢ−x̄)²]^(3/2)
첨도 = Σ(xᵢ−x̄)⁴ / [Σ(xᵢ−x̄)²]²
왜도
0 = 대칭
>0 = 오른쪽 꼬리
<0 = 왼쪽 꼬리
첨도
3 = 정규분포
>3 = 뾰족
<3 = 평편
E[(y−ŷ)²] = Bias² + Variance + Noise

복잡도↑ → Bias↓, Variance↑ (Trade-off)

MLE / MAP
우도 L(θ;X) = Πᵢ p(xᵢ|θ) [i.i.d.]
로그우도 ℓ(θ) = Σᵢ log p(xᵢ|θ)
MLE θ̂ = argmax ℓ(θ)
MAP θ̂ = argmax [ℓ(θ) + log P(θ)]

가우시안 MLE / LDA 결정 경계
P(x|Cₖ) = 1/√(2πσ²) · exp(−(x−μₖ)²/2σ²)
로그우도비 = 0 → x*(μ₁−μ₂)/σ² = (μ₁²−μ₂²)/2σ²
결정 경계 x* = (μ₁+μ₂)/2

x² 항 상쇄 → 선형 경계 / 공통 σ²: (Σ편차²₁+Σ편차²₂)/(n₁+n₂)

정규분포 & CLT
N(x;μ,σ²) = 1/√(2πσ²) · e^(−(x−μ)²/2σ²)
CLT: X̄ₙ → N(μ, σ²/n) as n→∞ (n≥30)

68-95-99.7 규칙: P(μ±σ)≈68%, P(μ±2σ)≈95%

Categorical MLE — 라그랑주 승수법
ℓ = Σ nₖ log πₖ, 제약: Σπₖ = 1
L(π,λ) = ℓ − λ(Σπₖ − 1)
1 ∂L/∂πₖ = 0 → πₖ = nₖ/λ
2 제약 Σπₖ=1 대입 → λ = n
3 π̂ₖ = nₖ/n (경험적 확률 = Softmax 수렴값)
✦ 연습문제
Q1
X~N(70,100). P(60≤X≤80)과 P(X>85)를 구하라. (Φ(1.5)≈0.9332)
P(60≤X≤80)≈68% / P(X>85)=1−0.9332≈6.7%
Q2
C₁ 40개, C₂ 60개. Categorical MLE로 π̂₁, π̂₂를 구하라.
π̂₁=0.4, π̂₂=0.6
Q3
μ₁=2, μ₂=6, σ²=4(공통)일 때 LDA 결정 경계 x*는?
x* = (2+6)/2 = 4
CH 04-05 선형 회귀 & 로지스틱 회귀
선형 회귀 (MSE 최소화)
모델 ŷ = θ₁x + θ₀
MSE J(θ) = (1/N) Σ(θ₁xᵢ+θ₀−yᵢ)²
∂J/∂θ₀ = 0 → θ₁Σxᵢ + Nθ₀ = Σyᵢ
∂J/∂θ₁ = 0 → θ₁Σxᵢ² + θ₀Σxᵢ = Σxᵢyᵢ
정규방정식 θ = (XᵀX)⁻¹Xᵀy
볼록 함수
편미분=0이
전역 최솟값 보장
성질
회귀 직선은
항상 (x̄, ȳ) 통과
로지스틱 회귀
σ(z) = 1/(1+e⁻ᶻ), σ'(z) = σ(z)(1−σ(z))
BCE L = −[y log ŷ + (1−y) log(1−ŷ)]
∂L/∂z = ŷ − y ← 시그모이드+로그 상쇄
∂J/∂θ₀ = (1/N) Σ(ŷᵢ−yᵢ)
∂J/∂θ₁ = (1/N) Σ xᵢ(ŷᵢ−yᵢ)
θ ← θ − α·∂J/∂θ [경사하강법]

초기 θ=0 → 모든 ŷ=0.5 → J=log2≈0.693 (최대 불확실성)

✦ 연습문제
Q1
θ₀=−3, θ₁=1일 때 x=3에서 로지스틱 예측값 ŷ는?
z=1·3+(−3)=0, ŷ=σ(0)=0.5
Q2
BCE 손실: y=1, ŷ=0.9일 때 손실 L은?
L=−log(0.9)≈0.105 (예측이 좋으면 손실 작음)
Q3
θ₀=−6, θ₁=1, θ₂=1일 때 결정 경계 2D 직선은?
x₁+x₂=6 (θᵀx=0 조건)
