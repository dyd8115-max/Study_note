CH01 · 컴파일러 구조
전단부 (Front-end)언어 의존적 · 분석 담당
후단부 (Back-end)기계 의존적 · 합성 담당
중간 코드 (IC)기계 독립적 중간 표현
심볼 테이블식별자 정보 · 모든 단계 공유
리터럴 테이블상수값 저장 · 중복 방지

IC 예시: Java 바이트코드, LLVM IR, Python .pyc

CH01 · 컴파일 단계 (6단계)
  • 어휘 분석 (Scanner) — 소스 → 토큰
    a[index] = 4+2 → id1 := id2 + id3 * 60
  • 구문 분석 (Parser) — 토큰 → 구문 트리
    AST 생성 (불필요한 기호 제거)
  • 의미 분석 (Semantic) — AST → 주석 트리
    타입 정보 추가, inttoreal 삽입
  • 중간 코드 생성 — 주석 트리 → IC
    3주소 코드: t = a op b
  • 코드 최적화 — 상수 접기 등
    4+2 → 6 (컴파일 시점에 미리 계산)
  • 코드 생성 — IC → 기계어
    레지스터 할당, 주소 지정
